## 1. 返回数组中符合条件的第一个元素
### 需求
1. 返回数组中符合条件的第一个元素
2.  [1, 2, 3]，条件num === 2; 那么返回值应为 2。

### 思路
1. 利用array.filter()返回符合条件的数组；
2. 再返回数组的第一个

###　代码
```
function find(arr, func) {
		let _arr = [];
		_arr = arr.filter(val => {
			return func(val);
		})
		var num = _arr[0];
		return num;
	}
```

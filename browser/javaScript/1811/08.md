# js浅拷贝和深拷贝
## 简单的赋值,只是复制指针，而共享内存

例如：
```
let obj1 = {
    a: 1;
    b: 2
}
let obj2 = obj1;
// 改变b也会影响a
obj2.a = 3;
console.log(obj1.a) //3
```

## 最简单的拷贝
> 只能简单拷贝值类型（如：String，Number...），如果复制引用类型还是复制指针，共享内存，还是得需要深拷贝

```
let obj1 = {
    a: 1,
};
let obj2 = {
    a: 2,
    b: 3
};
for (let name in obj2) {
    obj1[name] = obj2[name]
}
```

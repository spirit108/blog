# vue 实例的常用属性和方法
## 常用属性
1. $data: 是我们new vue实例时传进去的data
```
export default {
    data() {
        return {
            num: 0
        };
    },
    created() {
        console.log(this)
        console.log(this.$data); // 输出的值： {num: 0}
    }
};
```

2. $pops: 父组件给子组件传值声明
